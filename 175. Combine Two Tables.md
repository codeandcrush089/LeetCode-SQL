# Problem 175: Combine Two Tables
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/combine-two-tables/description/)

```sql
SELECT
    p.firstName,
    p.lastName,
    a.city,
    a.state
FROM
    Person p
LEFT JOIN
    Address a ON p.personId = a.personId;
```
---
## üßê Query Explanation

* **`FROM Person p`**
  Starts with the `Person` table so **every person appears**, even if they have no address.

* **`LEFT JOIN Address a ON p.personId = a.personId`**
  Matches each person to their address.

  * If the person has an address ‚Üí city and state are filled.
  * If not ‚Üí city and state are **NULL**.

* **`SELECT p.firstName, p.lastName, a.city, a.state`**
  Shows the person's name along with their city and state.

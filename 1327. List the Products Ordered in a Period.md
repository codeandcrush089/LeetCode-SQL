# Problem 1327: List the Products Ordered in a Period
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
    p.product_name,
    T.total_units AS unit
FROM
    Products p
INNER JOIN
    (
        SELECT
            product_id,
            SUM(unit) AS total_units
        FROM
            Orders
        WHERE
            -- Filter orders specifically for February 2020
            order_date >= '2020-02-01' AND order_date <= '2020-02-29'
        GROUP BY
            product_id
        HAVING
            -- Filter to keep only products with 100 or more total units
            SUM(unit) >= 100
    ) AS T
ON
    p.product_id = T.product_id;
```

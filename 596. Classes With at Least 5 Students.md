# Problem 596: Classes With at Least 5 Students
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/classes-with-at-least-5-students/description/)

```sql
SELECT
    class
FROM
    Courses
GROUP BY
    class
HAVING
    -- Filter the groups where the count of students (rows) is 5 or more
    COUNT(student) >= 5;
```
---
## ðŸ§ Query Explanation

### **1. `FROM Courses`**

Reads course enrollment data from the `Courses` table.


### **2. `GROUP BY class`**

Groups all rows by **class**, so students are counted per class.


### **3. `COUNT(student) >= 5`**

Counts how many students are enrolled in each class.


### **4. `HAVING COUNT(student) >= 5`**

Keeps only classes with **5 or more students**.


### **5. `SELECT class`**

Returns the **class names** that meet the condition.

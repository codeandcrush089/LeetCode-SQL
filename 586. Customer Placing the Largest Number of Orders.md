# Problem 586: Customer Placing the Largest Number of Orders
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/customer-placing-the-largest-number-of-orders/)

```sql
SELECT
    customer_number
FROM
    Orders
GROUP BY
    customer_number  -- Group the rows by customer_number
ORDER BY
    -- Count the number of orders (rows) in each group and order by this count descending
    COUNT(order_number) DESC
LIMIT 1; -- Select only the top customer
```
---
## üßê Query Explanation

### **1. `FROM Orders`**

Uses the `Orders` table, where each row represents an order.


### **2. `GROUP BY customer_number`**

Groups all orders by each customer so orders are counted **per customer**.


### **3. `COUNT(order_number)`**

Counts how many orders each customer has placed.


### **4. `ORDER BY COUNT(order_number) DESC`**

Sorts customers by **number of orders**, highest first.


### **5. `LIMIT 1`**

Returns **only the customer with the most orders**.

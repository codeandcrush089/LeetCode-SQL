# Problem 1378: Replace Employee ID With The Unique Identifier
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
    EU.unique_id,
    E.name
FROM
    Employees AS E
LEFT JOIN
    EmployeeUNI AS EU ON E.id = EU.id;
```
---
## üßê Query Explanation

* **`FROM Employees AS E`**
  Starts with the **Employees** table so **every employee** appears in the output.

* **`LEFT JOIN EmployeeUNI AS EU ON E.id = EU.id`**
  Joins employee records with their **unique ID**, if available.

  * If the employee‚Äôs ID **exists** in `EmployeeUNI` ‚Üí `unique_id` is shown.
  * If it **does not exist** ‚Üí `unique_id` becomes **NULL** (because of LEFT JOIN).

* **`SELECT EU.unique_id, E.name`**
  Retrieves:

  * The employee‚Äôs **unique ID** (or NULL if not found)
  * The employee‚Äôs **name**

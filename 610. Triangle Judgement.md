# Problem 610: Triangle Judgement.md
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
    x,
    y,
    z,
    CASE
        -- Check if all three sides satisfy the Triangle Inequality Theorem
        WHEN x + y > z AND x + z > y AND y + z > x THEN 'Yes'
        ELSE 'No'
    END AS triangle
FROM
    Triangle;
```

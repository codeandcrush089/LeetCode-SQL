# Problem 1729: Find Followers Count
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
    user_id,
    COUNT(follower_id) AS followers_count
FROM
    Followers
GROUP BY
    user_id
ORDER BY
    user_id ASC;
```
---
## üßê Query Explanation


* **`FROM Followers`**
  Reads the follower relationships.

* **`COUNT(follower_id) AS followers_count`**
  Counts how many followers each user has.

* **`GROUP BY user_id`**
  Groups rows by user so the count is calculated per user.

* **`ORDER BY user_id ASC`**
  Sorts the result by user ID in ascending order.

# Problem 1693: Daily Leads and Partners
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
    date_id,
    make_name,
    COUNT(DISTINCT lead_id) AS unique_leads,
    COUNT(DISTINCT partner_id) AS unique_partners
FROM
    DailySales
GROUP BY
    date_id,
    make_name;
```

## üßê Query Explanation

* `date_id, make_name` ‚Üí Groups data by **date** and **company (make)**.
* `COUNT(DISTINCT lead_id)` ‚Üí Counts the **number of unique leads** per date & make.
* `COUNT(DISTINCT partner_id)` ‚Üí Counts the **number of unique partners** per date & make.
* `GROUP BY date_id, make_name` ‚Üí Ensures counts are calculated **separately for each date and make**.

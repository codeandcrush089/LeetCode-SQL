# Problem 620: Not Boring Movies
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/not-boring-movies/)

```sql
SELECT
    id,
    movie,
    description,
    rating
FROM
    Cinema
WHERE
    -- Condition 1: id must be odd. The modulo operator (%) checks for a remainder of 1 when divided by 2.
    id % 2 = 1
    -- Condition 2: description must not be 'boring'.
    AND description != 'boring'
ORDER BY
    -- Order the results by rating from highest to lowest.
    rating DESC;
```
---
## üßê Query Explanation

### **1. `FROM Cinema`**

Reads movie records from the `Cinema` table.


### **2. Filter conditions**

```sql
id % 2 = 1
AND description != 'boring'
```

* `id % 2 = 1` ‚Üí keeps movies with an **odd ID**.
* `description != 'boring'` ‚Üí excludes movies marked as **boring**.


### **3. `SELECT id, movie, description, rating`**

Returns the movie details and rating.


### **4. `ORDER BY rating DESC`**

Sorts movies by **highest rating first**.

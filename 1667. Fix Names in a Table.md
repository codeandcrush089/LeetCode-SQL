# Problem 1667: Fix Names in a Table
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/fix-names-in-a-table/description/)

```sql
SELECT
    user_id,
    -- Concatenate the first character (made uppercase) with the rest of the name (made lowercase)
    CONCAT(
        -- 1. Convert the first character to uppercase: 
        -- SUBSTRING(name, 1, 1) extracts the first character.
        -- UPPER() converts it to uppercase.
        UPPER(SUBSTRING(name, 1, 1)),
        -- 2. Convert the rest of the string to lowercase:
        -- SUBSTRING(name, 2) extracts the string starting from the second character.
        -- LOWER() converts the remaining string to lowercase.
        LOWER(SUBSTRING(name, 2))
    ) AS name
FROM
    Users
ORDER BY
    user_id;
```

## üßê Query Explanation

* `SUBSTRING(name, 1, 1)` ‚Üí Gets the **first letter** of the name.
* `UPPER(...)` ‚Üí Converts the **first letter to uppercase**.
* `SUBSTRING(name, 2)` ‚Üí Gets the **remaining part** of the name.
* `LOWER(...)` ‚Üí Converts the rest to **lowercase**.
* `CONCAT(...)` ‚Üí Joins both parts to form a **properly formatted name**.
* `ORDER BY user_id` ‚Üí Sorts results by **user ID**.

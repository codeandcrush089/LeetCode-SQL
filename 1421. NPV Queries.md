# Problem 1421: NPV Queries
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
  Queries.id,
  Queries.year,
  IFNULL(NPV.npv, 0) AS npv
FROM Queries
LEFT JOIN NPV
  USING (id, year);
```
---
## üßê Query Explanation

* `FROM Queries LEFT JOIN NPV` ‚Üí Keeps **all records from Queries**, even if no match in NPV.
* `USING (id, year)` ‚Üí Joins both tables on **id and year**.
* `IFNULL(NPV.npv, 0)` ‚Üí Replaces **NULL NPV values with 0**.
* `SELECT id, year, npv` ‚Üí Displays **query ID, year, and its NPV**.

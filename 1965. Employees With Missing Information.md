# Problem 1965: Employees With Missing Information
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/employees-with-missing-information/description/)


```sql
SELECT Employees.employee_id
FROM Employees
LEFT JOIN Salaries
  USING (employee_id)
WHERE Salaries.salary IS NULL
UNION ALL
SELECT Salaries.employee_id
FROM Salaries
LEFT JOIN Employees
  USING (employee_id)
WHERE Employees.name IS NULL
ORDER BY 1;
```
---
## üßê Query Explanation


* **First SELECT**

  * `Employees LEFT JOIN Salaries` ‚Üí Keeps all employees.
  * `WHERE Salaries.salary IS NULL` ‚Üí Finds employees **who have no salary record**.

* **Second SELECT**

  * `Salaries LEFT JOIN Employees` ‚Üí Keeps all salary records.
  * `WHERE Employees.name IS NULL` ‚Üí Finds salaries **assigned to non-existing employees**.

* `UNION ALL` ‚Üí Combines both missing cases **without removing duplicates**.

* `ORDER BY 1` ‚Üí Sorts output by **employee_id**.

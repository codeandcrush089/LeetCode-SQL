# Problem 627: Swap Sex of Employees
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/swap-sex-of-employees/description/)

```sql
UPDATE
    Salary
SET
    sex = CASE
        WHEN sex = 'm' THEN 'f'
        ELSE 'm'  -- This handles the case where sex = 'f'
    END;
```
---
## üßê Query Explanation

### **1. `UPDATE Salary`**

Targets all rows in the `Salary` table for modification.

### **2. `SET sex = CASE ... END`**

Uses a `CASE` expression to update the `sex` column:

* If `sex = 'm'` ‚Üí change it to `'f'`
* Otherwise (i.e., `sex = 'f'`) ‚Üí change it to `'m'`

# Problem 1440: Evaluate Boolean Expression
![Medium](https://img.shields.io/badge/Level-Medium-yellow)


```sql
SELECT Expressions.*,
  (
    CASE
      WHEN Expressions.operator = '<'
      AND LeftOperand.value < RightOperand.value THEN 'true'
      WHEN Expressions.operator = '>'
      AND LeftOperand.value > RightOperand.value THEN 'true'
      WHEN Expressions.operator = '='
      AND LeftOperand.value = RightOperand.value THEN 'true'
      ELSE 'false'
    END
  ) AS value
FROM Expressions
INNER JOIN VARIABLES AS LeftOperand
  ON (Expressions.left_operand = LeftOperand.name)
INNER JOIN VARIABLES AS RightOperand
  ON (Expressions.right_operand = RightOperand.name);
```

## üßê Query Explanation

* `FROM Expressions` ‚Üí Reads all logical expressions.
* `INNER JOIN VARIABLES AS LeftOperand` ‚Üí Gets the **value of the left variable**.
* `INNER JOIN VARIABLES AS RightOperand` ‚Üí Gets the **value of the right variable**.
* `CASE WHEN operator = '<' AND left < right THEN 'true'` ‚Üí Evaluates **less than** condition.
* `CASE WHEN operator = '>' AND left > right THEN 'true'` ‚Üí Evaluates **greater than** condition.
* `CASE WHEN operator = '=' AND left = right THEN 'true'` ‚Üí Evaluates **equal to** condition.
* `ELSE 'false'` ‚Üí Marks expression as **false if condition fails**.
* `AS value` ‚Üí Stores the **result of the expression** as `'true'` or `'false'`.

# Problem 2356: Number of Unique Subjects Taught by Each Teacher
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/number-of-unique-subjects-taught-by-each-teacher/description/)


```sql
SELECT
  teacher_id,
  COUNT(DISTINCT subject_id) AS cnt
FROM Teacher
GROUP BY 1;
```
---
## üßê Query Explanation

* `COUNT(DISTINCT subject_id)` ‚Üí Counts how many **unique subjects** each teacher teaches.
* `GROUP BY teacher_id` ‚Üí Groups rows by teacher to compute the count per teacher.

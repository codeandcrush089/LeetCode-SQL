```sql
SELECT DISTINCT
    author_id AS id
FROM
    Views
WHERE
    author_id = viewer_id
ORDER BY
    id ASC;
```

---
## üßê Query Explanation


### 1. `SELECT DISTINCT author_id AS id`

Selects **unique author IDs** and renames the column as **`id`** in the output.


### 2. `FROM Views`

Uses the **Views** table as the data source.


### 3. `WHERE author_id = viewer_id`

Filters records where the **author viewed their own content** (author and viewer are the same person).


### 4. `ORDER BY id ASC`

Sorts the result in **ascending order of `id`**.

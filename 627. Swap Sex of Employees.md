# Problem 627: Swap Sex of Employees
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
UPDATE
    Salary
SET
    sex = CASE
        WHEN sex = 'm' THEN 'f'
        ELSE 'm'  -- This handles the case where sex = 'f'
    END;
```
---
## üßê Query Explanation

### **1. `UPDATE Salary`**

Targets all rows in the `Salary` table for modification.

### **2. `SET sex = CASE ... END`**

Uses a `CASE` expression to update the `sex` column:

* If `sex = 'm'` ‚Üí change it to `'f'`
* Otherwise (i.e., `sex = 'f'`) ‚Üí change it to `'m'`

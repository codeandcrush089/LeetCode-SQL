# Problem 3475: DNA Pattern Recognition 
![Medium](https://img.shields.io/badge/Level-Medium-yellow?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/dna-pattern-recognition/description/)

```sql
SELECT
    sample_id,
    dna_sequence,
    species,
    dna_sequence LIKE 'ATG%' AS has_start,
    dna_sequence REGEXP 'TAA$|TAG$|TGA$' AS has_stop,
    dna_sequence LIKE '%ATAT%' AS has_atat,
    dna_sequence REGEXP 'GGG+' AS has_ggg
FROM Samples
ORDER BY 1;
```
---
## üßê Query Explanation

* `dna_sequence LIKE 'ATG%'`
  ‚Üí Checks if the DNA sequence **starts with** the start codon `"ATG"`.

* `dna_sequence REGEXP 'TAA$|TAG$|TGA$'`
  ‚Üí Checks if the sequence **ends with** one of the stop codons: `"TAA"`, `"TAG"`, or `"TGA"`.

* `dna_sequence LIKE '%ATAT%'`
  ‚Üí Finds whether the substring `"ATAT"` appears **anywhere** in the sequence.

* `dna_sequence REGEXP 'GGG+'`
  ‚Üí Checks for **a run of at least three G's** in a row (`GGG`, `GGGG`, etc.).

* `ORDER BY sample_id`
  ‚Üí Sorts the results by sample.

Each expression returns **1 (true)** or **0 (false)**.

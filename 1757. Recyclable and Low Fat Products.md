# Problem 1757: Recyclable and Low Fat Products
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
    product_id
FROM
    Products
WHERE
    low_fats = 'Y' AND recyclable = 'Y';
```

## üßê Query Explanation

* `FROM Products` ‚Üí Reads data from the **Products** table.
* `low_fats = 'Y'` ‚Üí Selects only **low-fat products**.
* `recyclable = 'Y'` ‚Üí Selects only **recyclable products**.
* `AND` ‚Üí Ensures **both conditions must be true**.
* `SELECT product_id` ‚Üí Returns only the **product IDs**.

# Problem 1741: Find Total Time Spent by Each Employee
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/find-total-time-spent-by-each-employee/description/)


```sql
SELECT
    event_day AS day,
    emp_id,
    SUM(out_time - in_time) AS total_time
FROM
    Employees
GROUP BY
    event_day,
    emp_id
ORDER BY
    event_day,
    emp_id;
```
---
## üßê Query Explanation

* `event_day AS day` ‚Üí Renames **event_day** column to **day**.
* `out_time - in_time` ‚Üí Calculates **working time for one entry**.
* `SUM(...)` ‚Üí Adds up **total working time per employee per day**.
* `GROUP BY event_day, emp_id` ‚Üí Groups data **by day and by employee**.
* `ORDER BY event_day, emp_id` ‚Üí Sorts results by **day first, then employee ID**.

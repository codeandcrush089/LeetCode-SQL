# Problem 1068 Product Sales Analysis I
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
    p.product_name,
    s.year,
    s.price
FROM
    Sales AS s
INNER JOIN
    Product AS p ON s.product_id = p.product_id;
```


## üßê Query Explanation

### 1. `SELECT p.product_name, s.year, s.price`

Selects the **product name**, **sales year**, and **price** for the result set.


### 2. `FROM Sales AS s`

Takes data from the **Sales** table and assigns it the alias **`s`**.


### 3. `INNER JOIN Product AS p`

Performs an **INNER JOIN** with the **Product** table and assigns it the alias **`p`**.


### 4. `ON s.product_id = p.product_id`

Defines the **join condition**, matching each sale with its corresponding product using `product_id`.


Returns **only those records where a matching product exists for a sale**, showing product name, year, and price together.

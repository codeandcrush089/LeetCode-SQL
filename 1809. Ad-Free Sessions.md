# Problem 1809: Ad-Free Sessions
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/ad-free-sessions/description/)

```sql
SELECT Playback.session_id
FROM Playback
LEFT JOIN Ads
  ON (
    Playback.customer_id = Ads.customer_id
    AND Ads.timestamp BETWEEN Playback.start_time AND Playback.end_time)
WHERE Ads.ad_id IS NULL;
```
---
## üßê Query Explanation

* `FROM Playback LEFT JOIN Ads` ‚Üí Joins **playback sessions with ads**, keeping **all playbacks**.
* `customer_id` match ‚Üí Links ads to the **same customer**.
* `Ads.timestamp BETWEEN start_time AND end_time` ‚Üí Checks if an **ad appeared during the session**.
* `WHERE Ads.ad_id IS NULL` ‚Üí Keeps only sessions **with NO ads shown**.
* `SELECT Playback.session_id` ‚Üí Returns only the **session IDs**.

# Problem 1141: User Activity for the Past 30 Days I
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
    -- Rename the date column as 'day' for the final output
    activity_date AS day,
    -- Count the number of unique user IDs for each day
    COUNT(DISTINCT user_id) AS active_users
FROM
    Activity
WHERE
    -- Filter 1: Activity must be on or before the end date
    activity_date <= '2019-07-27'
    AND
    -- Filter 2: Activity must be on or after the start date (30 days before 2019-07-27)
    activity_date > DATE_SUB('2019-07-27', INTERVAL 30 DAY)
    -- Alternative, simpler way to calculate the start date: activity_date >= '2019-06-28'
GROUP BY
    activity_date;
```
---
## üßê Query Explanation

### 1. `SELECT activity_date AS day`

Renames the column **`activity_date`** to **`day`** for the final output.


### 2. `COUNT(DISTINCT user_id) AS active_users`

Counts the number of **unique users** active on each day and labels it as **`active_users`**.


### 3. `FROM Activity`

Uses the **Activity** table as the data source.


### 4. `WHERE activity_date <= '2019-07-27'`

Filters records to include only activities that occurred **on or before 27 July 2019**.



### 5. `AND activity_date > DATE_SUB('2019-07-27', INTERVAL 30 DAY)`

Filters records to include only activities that occurred **within the last 30 days before 27 July 2019**.

### 6. `GROUP BY activity_date`

Groups records by **each activity date** so that active users are counted **per day**.

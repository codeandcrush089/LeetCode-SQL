# Problem 1777: Product's Price for Each Store
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
  product_id,
  MAX(CASE WHEN store = 'store1' THEN price END) AS store1,
  MAX(CASE WHEN store = 'store2' THEN price END) AS store2,
  MAX(CASE WHEN store = 'store3' THEN price END) AS store3
FROM Products
GROUP BY 1;
```

## üßê Query Explanation

* `CASE WHEN store = 'store1' THEN price END` ‚Üí Picks the **price only for store1** (NULL for others).
* `MAX(...) AS store1 / store2 / store3` ‚Üí Converts **row values into separate columns** (pivoting).
* `GROUP BY 1` ‚Üí Groups by the **first selected column (product_id)**.
* `SELECT product_id` ‚Üí Shows prices **per product** across different stores.

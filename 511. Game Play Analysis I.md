# Problem 511: Game Play Analysis I
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/game-play-analysis-i/description/)

```sql
SELECT
    player_id,
    MIN(event_date) AS first_login
FROM
    Activity
GROUP BY
    player_id
ORDER BY
    player_id;
```
---
## üßê Query Explanation

### **1. `FROM Activity`**

Uses the `Activity` table, which stores player login activity.



### **2. `MIN(event_date) AS first_login`**

Finds the **earliest login date** for each player.



### **3. `GROUP BY player_id`**

Groups all activity records by player so the minimum date is calculated **per player**.



### **4. `SELECT player_id`**

Returns each player‚Äôs ID along with their first login date.



### **5. `ORDER BY player_id`**

Sorts the output by player ID.

# Problem 1795: Rearrange Products Table
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

```sql
SELECT
  product_id,
  'store1' AS store,
  store1 AS price
FROM Products
WHERE store1 IS NOT NULL
UNION ALL
SELECT
  product_id,
  'store2',
  store2
FROM Products
WHERE store2 IS NOT NULL
UNION ALL
SELECT
  product_id,
  'store3',
  store3
FROM Products
WHERE store3 IS NOT NULL;
```

## üßê Query Explanation

* Each `SELECT` picks prices from **one store column** (`store1`, `store2`, `store3`).
* `'store1' / 'store2' / 'store3' AS store` ‚Üí Adds the **store name as a value**.
* `WHERE storeX IS NOT NULL` ‚Üí Keeps only **available prices**.
* `UNION ALL` ‚Üí **Stacks all rows together** without removing duplicates.
